Objeto Implícito JSP

En JSP, hay varios objetos que puedes usar directamente sin declararlos. Estos son conocidos como objetos implícitos.

Objeto Implícito Tipo (clase/interface) Descripción
request javax.servlet.http.HttpServletRequest Requisición actual. Contiene información de la solicitud del cliente.
response javax.servlet.http.HttpServletResponse Respuesta al cliente. Puedes enviar datos al cliente.
session javax.servlet.http.HttpSession Proporciona una forma de identificar al usuario en múltiples solicitudes de página.
out javax.servlet.jsp.JspWriter Envía contenido a la salida del cliente.
application javax.servlet.ServletContext Comparte información entre todas las páginas de la aplicación web.
config javax.servlet.ServletConfig Proporciona información de configuración para la página JSP.
pageContext javax.servlet.jsp.PageContext Proporciona métodos para acceder a varios objetos de alcance.
exception java.lang.Throwable Gestiona excepciones que se produzcan en la página JSP (usado principalmente en páginas de error).

Directivas JSP

Las directivas proporcionan mensajes al contenedor JSP e influyen en la fase de traducción de la página JSP. Aquí algunas de las directivas más comunes:

Directiva Descripción Atributos principales
page Define atributos para la página JSP, como importar paquetes, tipo de contenido y manejo de errores. import, contentType, isErrorPage, language...
include Incluye el contenido de otro archivo en la página JSP en tiempo de traducción. file
taglib Declara una biblioteca de etiquetas personalizada y le da un prefijo para su uso en la página. uri, prefix

Elementos JSP

Los elementos JSP permiten incrustar código Java dentro de una página JSP. Estos son algunos de los elementos clave:

Elemento Sintaxis Descripción
Declaraciones <%! ... %> Se utiliza para declarar variables y métodos que se utilizan en el código JSP.
Scriptlets <% ... %> Contiene código Java que se ejecuta cada vez que la página se solicita.
Expresiones <%= ... %> Evaluadas y convertidas a String, y luego insertadas en la salida de la página.
Directivas <%@ ... %> Proporciona mensajes al contenedor JSP, afectando la fase de traducción.
Comentarios <!-- ... --> Comentarios que no se envían al cliente en la salida.
