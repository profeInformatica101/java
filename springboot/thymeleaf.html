Etiquetas más comunes de Thymeleaf

th:if permite mostrar u ocultar elementos HTML en función de una condición. 
<p th:if="${condition}">Este párrafo se muestra si 'condition' es verdadero.</p>
th:unless es el opuesto de th:if, y se muestra cuando la condición es falsa. 
<p th:unless="${condition}">Este párrafo se muestra si 'condition' es falso.</p>

th:each se utiliza para iterar sobre una colección y generar elementos HTML para cada elemento de la colección. 
<ul>
    <li th:each="item : ${items}" th:text="${item}">Elemento de la lista</li>
</ul>
Ejemplo: Si items = ["Uno", "Dos", "Tres"], se generará una lista con tres elementos: Uno, Dos, Tres.

th:object y th:field se utilizan para vincular campos de un formulario a propiedades de un objeto. 
<form th:object="${user}" th:action="@{/guardar}" method="post">
    <label for="name">Nombre:</label>
    <input type="text" id="name" th:field="*{name}" />
    <label for="email">Email:</label>
    <input type="text" id="email" th:field="*{email}" />
    <button type="submit">Guardar</button>
</form>

th:value se utiliza para establecer el valor de un atributo value en un elemento input o similar. 
<input type="text" th:value="${inputValue}" placeholder="Escriba algo aquí">
Ejemplo: Si inputValue = "Hola", el campo de texto tendrá el valor "Hola".

th:text se utiliza para establecer el contenido de texto de un elemento HTML. 
<p th:text="${message}">Este texto será reemplazado por el contenido de 'message' si está presente.</p>
Ejemplo: Si message = "Hola Mundo", el resultado será un párrafo que contiene "Hola Mundo".

th:href se utiliza para definir la URL de un enlace. 
<a th:href="@{/pagina}">Enlace a la página</a>
Ejemplo: Si la ruta es /pagina, el enlace apuntará a /pagina.

th:selected se utiliza para seleccionar automáticamente una opción en un elemento select cuando se cumple una condición. 
<select th:field="*{selectedOption}">
    <option value="opcion1" th:selected="${selectedOption == 'opcion1'}">Opción 1</option>
    <option value="opcion2" th:selected="${selectedOption == 'opcion2'}">Opción 2</option>
    <option value="opcion3" th:selected="${selectedOption == 'opcion3'}">Opción 3</option>
</select>

th:class se utiliza para agregar clases CSS a un elemento HTML en función de una condición. 
<div th:class="${condition} ? 'clase-activa' : 'clase-inactiva'">Elemento con clase dinámica</div>

th:style se utiliza para aplicar estilos CSS a un elemento HTML en función de una condición. 
<div th:style="${condition} ? 'color: red;' : 'color: green;'">Texto con estilo dinámico</div>
