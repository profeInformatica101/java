Términos Clave de Seguridad de Aplicaciones en Spring Boot

Identidad:
Un actor único, típicamente un individuo conocido como usuario.
// Ejemplo: UserDetails userDetails = new User("username", "password", authorities);
Credenciales:
Usualmente un ID de usuario y contraseña.
// Ejemplo: Authentication auth = new UsernamePasswordAuthenticationToken("username", "password");
Autenticación:
Es cómo la aplicación verifica la identidad del solicitante.
// Ejemplo: Authentication result = authenticationManager.authenticate(auth);
Autorización:
¿Puede un usuario realizar una acción?
// Ejemplo: @PreAuthorize("hasRole('ADMIN')") public String metodoAdministrativo() { return "Acción administrativa realizada"; }
