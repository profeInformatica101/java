Guía de Etiquetas Thymeleaf

Etiqueta th:text

La etiqueta th:text se utiliza para establecer el contenido de texto de un elemento HTML. Es comúnmente usada para mostrar dinámicamente datos en la página.

Ejemplo:

<p th:text="${message}">Este texto será reemplazado por el contenido de 'message' si está presente.</p>

Etiqueta th:if

La etiqueta th:if permite mostrar u ocultar elementos HTML en función de una condición. Si la condición es verdadera, el elemento se muestra; de lo contrario, se oculta.

Ejemplo:

<p th:if="${condition}">Este párrafo se muestra si 'condition' es verdadero.</p>

Etiqueta th:unless

La etiqueta th:unless es el opuesto de th:if. Permite mostrar elementos solo si la condición es falsa.

Ejemplo:

<p th:unless="${condition}">Este párrafo se muestra si 'condition' es falso.</p>

Etiqueta th:each

La etiqueta th:each se utiliza para iterar sobre una colección y generar elementos HTML para cada elemento de la colección.

Ejemplo:

<ul>
        <li th:each="item : ${items}" th:text="${item}">Elemento de la lista</li>
    </ul>

Etiqueta th:href

La etiqueta th:href se utiliza para definir la URL de un enlace.

Ejemplo:

<a th:href="@{/pagina}">Enlace a la página</a>

Etiqueta th:value

La etiqueta th:value se utiliza para establecer el valor de un atributo value en un elemento input o similar.

Ejemplo:

<input type="text" th:value="${inputValue}" placeholder="Escriba algo aquí">

Etiqueta th:object y th:field (para formularios)

Las etiquetas th:object y th:field se utilizan para vincular campos de un formulario a propiedades de un objeto.

Ejemplo:

<form th:object="${user}" th:action="@{/guardar}" method="post">
      <label for="name">Nombre:</label>
      <input type="text" id="name" th:field="*{name}" />
      <label for="email">Email:</label>
      <input type="text" id="email" th:field="*{email}" />
      <button type="submit">Guardar</button>
    </form>

Etiqueta th:selected

La etiqueta th:selected se utiliza para seleccionar automáticamente una opción en un elemento select cuando se cumple una condición.

Ejemplo:

<select th:field="*{selectedOption}">
            <option value="opcion1" th:selected="${selectedOption == 'opcion1'}">Opción 1</option>
            <option value="opcion2" th:selected="${selectedOption == 'opcion2'}">Opción 2</option>
            <option value="opcion3" th:selected="${selectedOption == 'opcion3'}">Opción 3</option>
        </select>

Etiqueta th:class

La etiqueta th:class se utiliza para agregar clases CSS a un elemento HTML en función de una condición.

Ejemplo:

<div th:class="${condition} ? 'clase-activa' : 'clase-inactiva'">Elemento con clase dinámica</div>

Etiqueta th:style

La etiqueta th:style se utiliza para aplicar estilos CSS a un elemento HTML en función de una condición.

Ejemplo:

<div th:style="${condition} ? 'color: red;' : 'color: green;'">Texto con estilo dinámico</div>
